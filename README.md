## usage
### requirements
- [nvim-treesitter](https://github.com/nvim-treesitter/nvim-treesitter)
### installation
```lua
use({
	"dariuscorvus/tree-sitter-surrealdb.nvim",
	requires = { "nvim-treesitter" },
})
```

### setup

```lua
require("tree-sitter-surrealdb").setup()
```
